# 📍 SmartCampus Navigator | دليل الحرم الجامعي الذكي

مشروع مبتكر يهدف إلى تسهيل حياة الطلاب داخل الحرم الجامعي، من خلال توجيههم إلى قاعات محاضراتهم بدقة وسرعة بناءً على بياناتهم الجامعية.

## 🚀 عن المشروع (About the Project)
كثير من الطلاب، خاصة المستجدين، يواجهون صعوبة في تحديد مواقع القاعات والجهات داخل الجامعة. **SmartCampus Navigator** هو حل تقني ذكي يقوم بتحليل الرقم الجامعي ورقم القاعة لتحديد الموقع بدقة (الكلية، الدور، والمدخل)، مما يوفر الوقت والجهد.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Python**: اللغة الأساسية للمشروع.
* **Streamlit**: لبناء واجهة مستخدم تفاعلية وسهلة الاستخدام.
* **Logic Programming**: لتحليل الأرقام الجامعية واستنتاج المواقع.
* **YOLO & LLMs** (مستقبلاً): لدمج الرؤية الحاسوبية والذكاء الاصطناعي في عمليات التوجيه.

## ✨ المميزات الرئيسية (Key Features)
* **تحديد الموقع الفوري**: بمجرد إدخال الرقم الجامعي ورقم القاعة.
* **دعم كليات متعددة**: يغطي كليات الهندسة، الحاسب، وغيرها.
* **واجهة بسيطة**: مصممة لتناسب احتياجات الطالب أثناء التنقل.
* **تحليل ذكي**: يعتمد على قواعد منطقية لربط البيانات بالمواقع الجغرافية داخل الحرم.

## ⚙️ كيف يعمل؟ (How it Works)
1. يقوم الطالب بإدخال **رقمه الجامعي**.
2. يقوم النظام بتحليل كود الكلية والقسم من الرقم.
3. يقوم الطالب بإدخال **رقم القاعة**.
4. يعرض النظام المسار الصحيح: (اسم المبنى -> الدور -> أقرب مدخل).

## 💡 رؤية مستقبلية (Future Roadmap)
* إضافة خرائط تفاعلية (Interactive Maps).
* دمج خاصية التعرف على الأماكن باستخدام الكاميرا (YOLO).
* توفير نسخة تطبيق للهواتف الذكية.
